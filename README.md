# TAOTATO Character Quiz

A quiz to determine what member of the Trinity and the One you are. Featuring: Cyclone, King, Spellbinder and Farrco.

## Requirements

- Ruby 2.6.6

## Setup

- `bundle install`
- `ruby main.rb`

And then go to http://localhost:4567/

## Testing

`bundle exec rspec`
